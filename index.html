1. 분석 개요

업무시설(기타 건축물)의 공정 특성을 반영하여, 층 반복 WBS(골조→외장→벽체→MEP→내부마감→시운전→준공)를 구성하고 SS/FS/lag 관계를 포함한 네트워크 기반 동적 CPM을 수행합니다. 단축(crashing) 과정에서는 복수 임계경로(CP)가 동시에 존재할 수 있으므로, 모든 CP를 동시에 단축시키는 최소비용 조합을 매 스텝 산정하여 비용 누락을 방지합니다.

2. 산정 방법론

Methodology

층수/연면적/공사비/비작업일을 기반으로 WBS를 구성하고, 공식 검증 공기와 비교합니다.

WBS + CPM(SS/FS/lag) + Multi-CP Crashing + 시설물 공식 검증

    3. 비용 구조

    동적 다중 CP + 최소비용 단축

    단축 시점마다 CP를 재탐지하고, CP가 복수이면 최소비용 컷(min-cut)으로 단축 작업 세트를 선택합니다.

    간접비
