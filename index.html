4. CPM 산출식

(FS) w(j→i) = d_j + lag
(SS) w(j→i) = lag

ES_i = max( ES_j + w(j→i) )
EF_i = ES_i + d_i
Project Duration D = max(EF_i)

Backward:
LS_i = min( LS_k - w(i→k) )
Float_i = LS_i - ES_i
CP ⇔ Float_i ≈ 0

단축/슬라이더 변경 시 d_i가 변하므로 CPM 재계산 → CP 전이/중첩을 실시간 반영.

5. U-Curve 및 MC 도출식

Cost Slope: s_i = (C_i(crash)-C_i(normal)) / (d_i(normal)-d_i(crash))
Direct: C_dir(D) = C_base + Σ (d_i^N - d_i)·s_i
Indirect: C_ind(D) = D · c_ind/day
Total: C_total(D) = C_dir(D) + C_ind(D)

MC = argmin_D C_total(D) over simulationPoints
단축 구간은 “복수 CP”가 가능하므로, CP 서브그래프에서 1일 단축에 필요한 최소비용 공정 집합을 min-cut으로 선택하고 반복하여 포인트를 생성한다. 또한 단축 후 CPM 재계산에서 즉시 기간이 줄지 않는 구간(SS/lag/전이)도 발생할 수 있으므로, 포인트 생성은 stagnation 허용(안전장치)으로 조기 종료를 방지한다.
